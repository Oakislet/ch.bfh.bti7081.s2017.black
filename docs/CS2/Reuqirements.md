# Task 03 Requirements

## User Requirements
 - Überprüfung des Blutzuckerspiegels
 - Reagieren, sobald der Insulinspiegel aus der Norm ist
 - Abrufen des Systemstatus(Unbekannte Fehler, Batterie, Insulinvorrat, Glucoselevel)
 - Abrufen der Messwerte
 - Auffüllen/Aufladen von Insulin / Batterie
 
## System Requirements
 
### Functional Requirements

 - Blutzuckerspiegel messen
 - Kommunikation nach aussen
 - Sichere Fehlerbehandlung

### Non-Functional Requirements
 - Genaue Messwerte (Abweichung < 0.1%)
 - Robustheit
	- Temperatur (zwischen -20°C & 50°C)
 - Sichere Datenübertragung

### Domain Requirements
 - None
